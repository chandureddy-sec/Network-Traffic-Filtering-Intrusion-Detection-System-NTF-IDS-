Model Explanation

This system uses a time-series based detection approach inspired by LSTM behavior to analyze sequential packet patterns. It profiles standard flow rhythms allowing adaptive detection against undefined Zero-Day style shifts in structure.

Detection Logic

Analysis evaluates traffic through sliding windows dynamically scoring properties:

  • Feature Extraction: IP, protocol distributions, size variations, and communication frequency natively collected via Scapy.
  • Sliding Window Analysis: Historical baselines calculated instantly aggregating packets iteratively without memory leaks.
  • Anomaly Scoring System: Non-intrusive scaling producing dynamic risk percentage (0 to 100), triggering heuristic alert bands natively mapped back against network flow matrices.

Attack Analysis

For anomalous packets identified during this session, potential threats include definitions across standard vectors:

  • Port Scanning: Vertical or horizontal sweeps attempting service identification.
  • Flooding (DoS): Influx algorithms overloading destination architecture logic.
  • Abnormal UDP Traffic: Volumetric anomaly thresholds breached violating standard application protocol bounds.
  • Suspicious Protocol Behavior: Irregular sequence timing and size mutations.
